## Dataset

This project uses colorectal cancer data from the SEER (Surveillance, Epidemiology, and End Results) program.

Due to file size and access restrictions, the full dataset is not included in this repository.

To reproduce this analysis:
- Obtain SEER data via SEER*Stat
- Export the dataset as a CSV file
- Update the file path in the notebook accordingly

Note: A local file path is used in the notebook and should be modified based on your environment.
